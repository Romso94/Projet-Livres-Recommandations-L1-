Projet : Systeme de recommandation de livres  ( projet L1)
Auteurs : Gas Romain - Poscente Florian

GUIDE D'UTILISATION :


Ouverture :

-Ouvrir main.py avec un IDE puis l'éxéctuez.
-Une fentêtre d'accueil apparait.


Accueil :

-Vous avez 2 choix possibles  : S'enregistrer ou se connecter.

Enregistrement : 

-Cliquez sur le bouton "S'enregistrer".
-Entrez les données demandées puis cliquez sur le bouton en bas "S'enregistrer". 
-Vous allez être redirigé sur le page d'accueil.

Connexion :

-Une fois sur la page d'accueil, cliquez sur le bouton "Se connecter".
-Entrez les données, le pseudonyme et mot de passe rentrés doivent être le même que celui de l'enregistrement.
-Cliquez sur "Se connecter".
-Vous allez être dirigé sur votre espace.


Votre espace :

-3 options s'offrent à vous. 
-Pour commencer, cliquez sur "Votre Profil".

Votre Profil :

-Il est affiché vos données rentrées précédemment.
-Vous avez le choix entre "Modifier Mon Profil" ou "Supprimer Mon Profil".
-Cliquez sur "Modifier Mon Profil".

Modifier Votre Profil :

-Vous pouvez modifiez n'importe quel élément affiché à l'écran.
-Si vous décidez de modifier, changez les éléments qui ne vous conviennent pas puis cliquez sur "Modifier".
-Vous allez être déconnecté de votre espace et dirigé à l'accueil.
-Reconnectez-vous: si vous avez modifié votre pseudonyme, faites attention à bien mettre celui modifié et non l'ancien.


Supprimer Mon Profil :

-Retournez sur Votre Profil.
-Cliquez sur le bouton "Supprimer Mon profil".
-Si vous souhaitez supprimer votre profil, cliquez sur "supprimer" et votre profil sera supprimé.
-Attention à bien s'inscrire a nouveau et se connecter si le profil est supprimé.

Depot des livres :

-Une fois sur votre espace, cliquez  sur "Depot des livres".
-3 bouttons différents sont affichés.


Modifier le titre du livre :

-Séléctionnez le livre.
-Cliquez sur "Modifier le titre du livre".
-Choisissez un titre et cliquez sur "Modifier". Le titre du livre en question est modifié.
-Revenez sur votre Depot des livres.

Supprimer le livre :

-Séléctionnez un livre.
-Cliquez sur "Supprimer le livre". Le livres en question est supprimé.
-Revenez sur votre Depot des livres.

Ajouter un livre :

-Cliquez sur "Ajouter un livre".
-Entrez un nom de livre puis cliquez sur "Enregistrer". Le livre est ajouté.

Livres recommandés pour vous : 

-Sur votre espace, cliquez sur "Livres recommandés pour vous".
-Vous voyez un bouttons "Ajouter une note" : Cliquez dessus afin de mettre une note a un livre que vous avez lu
-Choississez la note, le livre et cliquer sur "Noter"
